uNav with offline maps!

It's the time ;)

You'll need to do a few manual steps
from a PC

Don't worry, they are not a big deal :)

  • Install the software

    You can download maps from any map server with any app!
    You can do it from the Terminal. Just install:

    ~$sudo apt-get install downloadosmtiles

  • Get the area

    Go to OpenStreetMap > Export > Manually select a different area > Choose your area
    You have the coordinates in step 4: lat1:lat2 & lon1:lon2

  • Download that area

    Use those coordinates in downloadosmtiles:

    ~$mkdir maps

    ~$cd maps

    ~$/usr/bin/downloadosmtiles --lat=51.4244:51.4913 --lon=6.9837:7.0422 --zoom=13:17

  • Move the maps to phone

    Move that maps folder to ~/.local/share/navigator.costales into the phone

  • Set offline maps

    Open uNav and go to Settings. Choose Offline maps:

  • Enjoy it!

    It's done |o/
