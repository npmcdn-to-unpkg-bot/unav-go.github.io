uNav with offline maps!

It's the time ;)

You'll need to do a few manual steps from an Ubuntu Desktop

Don't worry, they are not a big deal :)

  • Download the map tiles

    You can download maps from any map server with any app!
    You can do it from the Terminal. Just install the program:

    ~$sudo apt-get install downloadosmtiles

  • Get the area

    Tip: Go to OpenStreetMap and click on Export.
    You have the coordinates in step 4:

  • Download that area

    Use those coordinates in downloadosmtiles

    (Choose the zoom range you desire)

    ~$mkdir maps

    ~$cd maps

    ~$/usr/bin/downloadosmtiles --lat=51.4244:51.4913 --lon=6.9837:7.0422 --zoom=17:19

  • Move the maps to phone

    Move that maps folder to ~/.local/share/navigator.costales into the phone

  • Set offline maps

    Open uNav and go to Settings. Choose Offline maps:

  • Enjoy it!

    It's done |o/
